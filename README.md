**An industry-aligned roadmap designed to master Linux commands, networking, and automation.**

## üü¢ Phase 1: Linux Basics & Introduction
*Goal: Understand what Linux is and get it running.*

- [ ] **Introduction to Linux:** What is Linux? History and Open Source philosophy.
- [ ] **Linux vs. Unix:** Understanding the key differences and similarities.
- [ ] **Linux Architecture:** Hardware, Kernel, Shell, and User Space.
- [ ] **Distributions (Distros):** Choosing the right one (Ubuntu, Mint, CentOS, Kali).
- [ ] **Installation:** Setting up a Virtual Machine (VM), Dual Boot, or WSL.

## üü° Phase 2: Mastering the Command Line (CLI)
*Goal: Navigate the system without a mouse.*

- [ ] **Basic Commands:** `ls`, `cd`, `pwd`, `mkdir`, `rmdir`.
- [ ] **File Operations:** `cp`, `mv`, `rm`, `touch`.
- [ ] **Viewing Files:** `cat`, `less`, `more`, `head`, `tail`.
- [ ] **Searching & Filtering:** `grep`, `find`, `locate`.
- [ ] **Standard I/O:** Redirection (`>`, `>>`) and Pipes (`|`).

## üü† Phase 3: File System & Permissions
*Goal: Understand the "Linux way" of organizing data.*

- [ ] **File Hierarchy Standard (FHS):** Understanding `/etc`, `/bin`, `/var`, `/home`, etc.
- [ ] **File Permissions:** Understanding `rwx` (Read, Write, Execute) and Octal notation (755, 644).
- [ ] **Ownership:** Using `chmod` and `chown`.
- [ ] **Links:** Soft links (Symbolic) vs. Hard links.

## üî¥ Phase 4: Intermediate System Skills
*Goal: Manage the "brain" of the computer.*

- [ ] **Process Management:** `ps`, `top`, `htop`, `kill`, `nice`.
- [ ] **User & Group Management:** `useradd`, `groupadd`, `usermod`, `sudoers`.
- [ ] **Package Management:** Using `apt`, `yum`, or `dnf` to install software.
- [ ] **Disk Management:** `df`, `du`, `fdisk`, and mount points.
- [ ] **Archiving:** `tar`, `gzip`, `zip`.


## üõ°Ô∏è Phase 6: Networking & Security
*Goal: Connect and protect your Linux machine.*

- [ ] **Networking Tools:** `ping`, `ifconfig` (or `ip`), `netstat`, `ssh`, `scp`.
- [ ] **Firewalls:** Basics of `iptables` or `ufw`.
- [ ] **Logging:** Reading system logs in `/var/log`.
- [ ] **Encryption:** Understanding SSH keys and basic file encryption.
